# Queues

## Overview
FIFO (First In First Out) data structure

## Types
- Simple Queue
- Circular Queue
- Priority Queue
- Deque (Double-ended queue)

## Operations
- Enqueue: O(1)
- Dequeue: O(1)
- Peek: O(1)

## Common Uses
- BFS traversal
- Task scheduling
- Cache implementation (LRU)

## Practice Problems
- Implement queue using stacks
- Sliding window maximum
- Design circular queue
