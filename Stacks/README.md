# Stacks

## Overview
LIFO (Last In First Out) data structure

## Operations
- Push: O(1)
- Pop: O(1)
- Peek: O(1)

## Common Patterns
- Monotonic stack
- Valid parentheses
- Next greater/smaller element

## Applications
- Function call stack
- Undo mechanisms
- Expression evaluation
- DFS traversal

## Problems
- Valid parentheses
- Min stack
- Evaluate reverse polish notation
