# Linked Lists

## Overview
Linear data structure where elements are stored in nodes, each pointing to the next node.

## Types
- Singly Linked List
- Doubly Linked List
- Circular Linked List

## Common Operations
- Insert (beginning/end/middle): O(1)/O(n)
- Delete: O(n)
- Search: O(n)
- Reverse: O(n)

## Key Patterns
- Two pointer technique (fast/slow)
- Dummy node for edge cases
- Recursion for reversal

## Common Problems
- Reverse linked list
- Detect cycle
- Merge two sorted lists
- Find middle element
- Remove nth node from end
